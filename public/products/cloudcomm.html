Security

Enterprise-grade from day one

JWT + Refresh Token Rotation

15-minute access tokens. Rotating 7-day refresh tokens. Sessions invalidated server-side on logout.

bcrypt + Account Lockout

Passwords hashed with bcrypt. 5 failed logins triggers a 30-minute lockout automatically.

True Multi-Tenant Isolation

Every record scoped to its company. One tenant can never access another's data — at the API level.

Tamper-Proof Audit Log

Every action logged with user ID, role, IP, module, and timestamp. Immutable and searchable.

Dual-Layer Rate Limiting

1,000 req/15 min per IP + NestJS throttle. Login endpoint: 5 attempts per minute hard cap.

Helmet + CORS + SQL-Proof

Security headers, strict origin allowlist, and Prisma parameterized queries — injection impossible.
